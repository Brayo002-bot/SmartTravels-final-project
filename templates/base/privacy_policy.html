SmartTravels is explicitly committed to protecting your system identity and metadata. This documentation outlines how our infrastructure records, processes, and safeguards transactional logs across the transport booking layer.

01. Information We Collect

  • Name (Ticket Validation Only)
  • Phone Number (M-Pesa STK Gateway)
  • Itinerary (Routes, Seats, Timestamp)
  • GPS Telemetry (Active Journeys Only)

02. Why We Collect Your Data

We isolate data metrics transparently for the following performance dependencies:

  • To map system inventories and render validated boarding codes.
  • To safely handle transaction pipelines with mobile money operators.
  • To manage precise travel manifests required by security compliance and vehicle captains.
  • To correctly attribute programmatic loyalty tiers and fare credits to regular commuters.

03. Storage Requirements

Passengers can interact with booking engines without permanent baseline account overheads. Our database clusters cache lightweight parameters for the exact lifecycle required to track and certify an isolated checkout loop.

04. Data Sharing Scope

We maintain zero-monetization broker standards. Data values are uniquely relayed to primary fulfillment partners (bus operations, rail lines, regional airlines) strictly to guarantee seat check-in logistics.

05. Security Infrastructure

Sensitive parameters sit behind strict firewalls. System access configurations are heavily monitored using cryptographically signed SSL layers, restricted database access tokens, and localized threat mitigation engines.

06. Driver GPS Tracking

Fleet vehicles report location telemetry constantly for active operations safety optimization. Commuter geolocation triggers only match localized path checkpoints during active travel states and drop monitoring functions automatically when checking out at terminal structures.

07. Administrative Contact

If you require deep system clarification on localized data handling architectures, compliance states, or data removal paths, please file a support request through the central SmartTravels helpdesk.

Your interaction with our platform confirms your acknowledgment of these privacy measures.
